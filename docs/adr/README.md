# Architecture Decision Records (ADRs)

Architectural decisions with rationale, alternatives considered, and consequences. Each ADR records a specific choice between alternatives.

ADRs are organized by lifecycle phase in subdirectories:

- `Draft/` — investigation in progress.
- `Proposed/` — recommendation formed and ready for review.
- `Adopted/` — decision accepted and in effect.
- `Retired/` — decision superseded or no longer relevant.
- `Superseded/` — replaced by a newer ADR.

See [list-adrs.md](list-adrs.md) for the full lifecycle index.
