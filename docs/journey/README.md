# User Journeys

End-to-end experience maps showing how personas accomplish goals across the Porthole system. Each journey identifies stages, actions, satisfaction scores, and pain points that inform which Epics and Specs to create.

See `list-journeys.md` for the full index.

## Artifact format

Each journey lives in its own folder:

```
docs/journey/(JOURNEY-NNN)-<Title>/
  (JOURNEY-NNN)-<Title>.md   — primary journey narrative + Mermaid diagram
```

## Phases

```
Draft → Validated · Archived · Abandoned
```
